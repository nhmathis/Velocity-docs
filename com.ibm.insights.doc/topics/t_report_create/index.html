Create a report

Use the Home page to create one of the following reports.

  • Custom
  • Delivery Transformation
  • Recent Activity

  • Team Performance

  • From the Home page, click .

  • On the New report page select the type of report, type a name for the report, description, and select an icon to associate with the report.
  • Click Add. After the report is created you can immediately run, edit, and share the report with other users.
